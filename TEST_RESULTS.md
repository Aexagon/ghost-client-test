# Ghost client pipeline test — results (2026-07-05)

**Purpose:** validate the design-taste pipeline end-to-end on a fabricated client whose energy is the opposite of every real claimed direction (bright/loud/saturated vs the roster's calm/dark/earthy).

**Setup:** one agent invented "Dee" (SG hawker-food creator) and wrote raw voice transcripts + a sealed INTENT.md answer key. A second agent blind-derived a visual direction from the transcripts only. Judge (Fable, director) scored derivation vs key. Winner built into DEE_BRAND_IDENTITY.html.

**Score vs sealed key: pass on all five criteria.**
- Tone words: FUN / LOUD / HONEST recovered verbatim
- Palette temperature: bright + saturated + warm hit (flame #FF5C1A, chilli #E01B2E, turmeric #FFC61E, pandan pop #1E7F5C)
- Typography energy: poster-shout display + friendly body hit (Anton + Familjen Grotesk)
- All 3 "never" categories caught with verbatim quotes (atas-minimal, corporate polish, moody food-porn)
- Feeling sentence: near-identical imagery, independently reached (wet-market morning, loud friend pulling you in)

**What this proves, with the real-client blind test (2026-07-04):**
- Real-client test: pipeline recovers a warm/calm voice → reproduced that client's shipped design system from raw audio transcripts alone
- Dee test: pipeline recovers a loud/bright voice → correctly INVERTED house instincts (white space treated as the enemy) instead of pattern-matching the library
- Together: the colour-by-feeling logic listens to evidence rather than defaulting
