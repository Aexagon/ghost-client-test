# Changelog

Small repo, honest log. What changed and the reason it earned its place.

## 2026-07-24

- **The method now installs as an Agent Skill.** New [skills/design-from-voice/SKILL.md](skills/design-from-voice/SKILL.md), one file carrying the interview gate, the derivation discipline, and the exit checks, discoverable by `npx skills add`. The copy-paste prompts remain the way in for plain chat.
- **New derive rule: defaults are not evidence.** Thin evidence is where an AI reaches for its comfort palettes (beige and brass for anyone artisan, purple glow for anyone tech, sage for anyone calm). The reach is now banned by name unless a quote demands it. Learned from watching rules-based skills fight the same defaults from the output side; here it gets fought at the source.
- **New third check: the swap test.** If the direction would survive a different person's transcripts, it designed the niche, not the person. Redo from their quotes.
- **The Dee identity is now visible in the README.** The proof run was buried as raw HTML; a rendered image now shows what the method produced, with every element citing a line Dee said.

## 2026-07-11

- First public shape: QUESTIONS, INTERVIEW_PROMPT with the coverage gate, DERIVE_PROMPT, APPLYING_IT, and the Dee blind-test run in example/. Follow-the-heat rule hardened after a live interview test.
